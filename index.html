Projects

Custom Macropad

November 2025



This is a custom macropad featuring eight keys and two (sadly nonfunctional) LEDs. It runs off of a XIAO RP2040 devboard, and I created a custom PCB, CAD 3D printed enclosure, and KMK firmware. Currently, the keys are mapped to browser functions and copy / paste keys, but can be reprogrammed easily.

PCB Creation

The PCB was created in KICAD using a hackclub tutorial. I used the incorrect footprints for the LEDs, so they do not work. The switches were wired in a 2x8 matrix to save on GPIO pins. I used Affinity Designer to create the custom silkscreen design.


CAD

I created the case model in Fusion 360, and went through two iterations. At first, I printed the body too short, so I increased the lid height and was able to fit it all together with the M3 screws and heat-set inserts provided.

Putting it All Together

Soldering all of the switches, diodes, and the devboard was a process. This was my first time soldering, so I had some trouble, but I learned quickly. I created a quick script using CircuitPython and the KMK library, and my keyboard was functional!
